# 📖 Discogs Clone <a name="about-project"></a>

> This is a basic version of Discogs in which the user can add music records, delete them, view them, filter them, sort them, and search for titles or artists.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
- Git
- Ruby
- Ruby on rails gem `gem install rails`

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:ha-manel/Discogs-Clone-API.git
```

### Install

Install this project with:

```sh
  cd my-project
  bundle install
  rails db:create
  rails db:migrate
  rails db:seed
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```

### Run tests

To run tests, run the following command:

```sh
  rspec 
```
